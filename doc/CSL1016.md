# CSL1016: AsyncEventHandlerAttribute is missing argument

The `AsyncEventHandler` attribute must have at least one argument or no arguments at all. An empty list of arguments doesn't make sense.

## Sample code

```cs
[AsyncEventHandler()] // CSL1016: AsyncEventHandlerAttribute is missing argument.
public async Task FooAsync()
{
    await Task.Delay(0);
}
```

## Compliant code

```cs
[AsyncEventHandler]
public async Task FooAsync()
{
    await Task.Delay(0);
}
```
