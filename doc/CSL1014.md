# CSL1014: Consider using `<inheritdoc />`

Diagnoses when some code documentation is duplicated.

Consider the following declarations, where a class overrides a virtual method:

```cs
class BaseFoo
{
    /// <summary>
    /// Performs application-defined tasks associated with freeing, releasing, or resetting unmanaged resources.
    /// </summary>
    protected virtual void Dispose(bool disposing)
    {
        /* ... */
    }
}

class Foo : BaseFoo
{
    /// <summary>
    /// Performs application-defined tasks associated with freeing, releasing, or resetting unmanaged resources.
    /// </summary>
    protected override void Dispose(bool disposing)
    {
        base.Dispose(disposing);
        /* ... */
    }
}
```

The summary is a copy of the `BaseFoo.Dispose` doc. It may be preferable to use the `inheritdoc` tag instead:

```cs
class BaseFoo
{
    /// <summary>
    /// Performs application-defined tasks associated with freeing, releasing, or resetting unmanaged resources.
    /// </summary>
    protected virtual void Dispose(bool disposing)
    {
        /* ... */
    }
}

class Foo : BaseFoo
{
    /// <inheritdoc />
    protected override void Dispose(bool disposing)
    {
        base.Dispose(disposing);
        /* ... */
    }
}
```

The [CSL1017](CSL1017.md) diagnostic is also issued when implementing an interface.

See the [inheritdoc](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/xmldoc/recommended-tags#inheritdoc) section for replacing or overriding documentation.

## Sample code

```cs
class BaseFoo
{
    /// <summary>
    /// Performs application-defined tasks associated with freeing, releasing, or resetting unmanaged resources.
    /// </summary>
    protected virtual void Dispose(bool disposing)
    {
        /* ... */
    }
}

class Foo : BaseFoo
{
    /// <summary>
    /// Performs application-defined tasks associated with freeing, releasing, or resetting unmanaged resources.
    /// </summary>
    protected override void Dispose(bool disposing) // CSL1014: Consider using the <inheritdoc/> tag to document 'Dispose'
    {
        base.Dispose(disposing);
        /* ... */
    }
}
```
